# Global-Economic-and-Demographic-Dashboard

## Project Overview
This is an end-to-end Data Analytics project where I used Excel, SQL, and Power BI to clean, transform, analyze, and visualize business data.
- The project demonstrates the complete analytics workflow:
- Data Cleaning
- Data Transformation
- Data Analysis using SQL
- Data Visualization using Power BI
- Business Insight Generation

## Tools & Technologies Used

- Microsoft Excel – Data cleaning & preprocessing
- MySQL (SQL) – Data querying & analysis
- Power BI – Dashboard creation & visualization
- DAX – Calculated measures & KPIs

## Key Insights

- Identified top-performing products
- Analyzed sales trends over time
- Compared regional performance
- Discovered high-revenue customer segments

## Conclusion

This end-to-end project showcases my ability to clean, analyze, and visualize data using Excel, SQL, and Power BI. It strengthened my skills in data processing, querying, and dashboard creation, helping me turn raw data into meaningful business insights.
